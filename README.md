# Codeply

A modern, interactive coding platform designed to help developers learn, practice, and showcase their coding skills.

## Overview

Codeply is a web-based application built with HTML and JavaScript that provides an intuitive interface for writing, testing, and executing code. Whether you're a beginner learning to code or an experienced developer looking to practice, Codeply offers a seamless coding experience.

## Features

- 🎯 **Interactive Code Editor** - Write and edit code with syntax highlighting
- ⚡ **Live Code Execution** - Run your code instantly and see results in real-time
- 💾 **Save & Share** - Store your projects and share them with others
- 🎨 **Clean Interface** - Minimalist and user-friendly design
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🚀 **Fast Performance** - Optimized for quick load times and smooth interactions

## Tech Stack

- **Frontend**: HTML5 & CSS3
- **Scripting**: JavaScript (ES6+)
- **Architecture**: Client-side web application

## Language Composition

- HTML: 95.9%
- JavaScript: 4.1%

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for online features

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AwaisSDev/Codeply.git
```

2. Navigate to the project directory:
```bash
cd Codeply
```

3. Open the application in your web browser:
- Simply open `index.html` in your default browser, or
- Use a local development server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Usage

1. **Write Code** - Enter your code in the editor
2. **Execute** - Click the run button to execute your code
3. **View Results** - See output and results in real-time
4. **Save Project** - Save your work locally or in the cloud
5. **Share** - Share your projects with others via link or embed
